Hi all,

for the next star devlopment cycle, I like to start a discussion 
on archive file formats. This includes:

-	A new sparse file format that is closer to the POSIX.1-2001
	extensions and thus more compat than the current format.

-	the NFSv4 ACL format

-	the format to store NFSv4 extended attribute files

-	the format to store ZFS (and maybe more general) file attributes
	that are bejond the POSIX meta data.

I am Cc:ing to opensolaris-code in hope to get interest to more people than 
just those who are on the star development lists. I plan to remove 
opensolaris-code after the set of interested people has been set up.

Please feel free to start with introducing comments.
